# React template for Echarts

## intro
React template based on Echarts(An Open Source JavaScript Visualization Library)

You can learn more from [echarts](https://echarts.apache.org/en/index.html), or [echarts-for-react](https://git.hust.cc/echarts-for-react/)

We recommend using method of echarts, which provides more detailed docs, as well as richer examples.
